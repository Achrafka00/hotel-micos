# Micos Hotel Management System

Welcome to the **Micos Hotel Management System**, a web application designed to streamline hotel management processes, including room management, bookings, gallery uploads, and customer communication. This project is built using the Laravel framework, offering a robust and scalable solution for hotel operations.

---

## About the Project

The Micos Hotel Management System provides the following features:

- **Room Management**: Add, update, and delete room details, including pricing, descriptions, and amenities.
- **Booking Management**: Manage customer bookings, approve or reject reservations, and track booking statuses.
- **Gallery Management**: Upload and manage images of the hotel to showcase rooms and facilities.
- **Customer Communication**: Send personalized emails to customers with custom greetings, body text, and action links.
- **Contact Form**: Allow customers to send inquiries directly to the hotel.

This project is developed and maintained by **Achraf**.

---

## Features

### Admin Features
- Create, update, and delete rooms.
- Approve or reject customer bookings.
- Manage hotel gallery images.
- View and respond to customer messages.
- Send custom emails to customers.

### Customer Features
- Browse available rooms.
- View the hotel gallery.
- Submit booking requests.
- Contact the hotel via the contact form.

## Technologies Used

- **Backend**: Laravel (PHP Framework)
- **Frontend**: Blade Templates, Bootstrap
- **Database**: MySQL
- **Email**: Gmail SMTP for sending emails
- **Other Tools**: Laravel Notifications, Laravel Migrations

## Installation

Follow these steps to set up the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/hotel_project.git
   cd hotel_project
2. Install dependencies:
    composer install
    npm install
3. Set up the .env file:
    Copy the .env.example file to .env:
        cp .env.example .env
        Update the database and email configuration

4. Generate the application key:
    php artisan key:generate

5. Run migrations to set up the database:
    php artisan migrate

6. Serve the application:
    php artisan serve

7. Access the application in your browser:
    http://localhost:8000
